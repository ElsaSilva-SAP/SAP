## ðŸ’¡ Part 2: What is a Project?

### Simple Definition

**Definition:** A project is a temporary endeavor with a specific goal, timeline, and resources.

**In Simple Terms:** A project is work that has:
- A clear goal
- A start and end date
- Specific resources (people, money, time)
- A defined outcome

**Examples:**
- Building a house
- Organizing an event
- Implementing SAP
- Launching a new product

**Your Examples:**
- ___________________________________________
- ___________________________________________

---

### Project vs. Regular Work

**Regular Work:**
- Ongoing
- No end date
- Routine tasks
- Example: Daily operations

**Project:**
- Temporary
- Has end date
- Unique goal
- Example: Implementing new system
